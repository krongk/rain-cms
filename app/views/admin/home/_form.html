如何添加表单功能

表单在程序中有一个表(admin_comemnts)对应两个Model(comment, admin_comment) comment Model对应前端
admin_comment Model 对应后台
要添加表单，需要创建一个模版文件：temp_comment.html
temp_comment.html 的结构大致如下：

    <%= render file: 'public/templetes/unify/_header.html' %>

    <div class="container">   
      <!-- Comment Boxes -->
      <%= yield(:comment) %>
    </div>

    <%= render file: 'public/templetes/unify/_footer.html' %>
最重要的是要加： yield(:comemnt)

在前台添加表单样式

  <%= simple_form_for(@comment, html: { class: 'footer-subsribe' } ) do |f| %>                

    <div class="input-group">
        <input type="text" class="form-control" name="comment[mobile_phone]" placeholder="请输入您的手机号">                            
        <span class="input-group-btn">
            <button class="btn-u" type="submit">立即预定</button>
        </span>
    </div>         
  <% end %>
