安装说明

RainCMS安装步骤：

  1. git clone git@github.com:krongk/rain-cms.git your_dir
  2. cp config/application.example.yml config/application.yml
  3. vi config/application.yml
  4. cp config/database.example.yml config/database.yml
  5. vi config/database.yml
  6. vi db/seeds.rb
  7. bundle install
  8. bundle exec rake db:create
  9. bundle exec rake db:migrate
  10. bundle exec rake db:seed

* 也可以定制config/deploy.rb， 通过capistrano进行部署
