辅助方法汇总

方法说明
get_date(date) data: Date类型
返回时间格式：date.strftime("%Y-%m-%d")
get_short_content(content, count = 120) content: 字符串类型
返回content的前半截(120个字符)内容，往往用于显示文章摘要。
get_url(obj) obj: Admin::Page或Admin::Channel的实例
返回给定实例对应的URL地址
get_next_obj(obj) obj: Admin::Page或Admin::Channel的实例
返回给定实例的下一个实例
get_menu(short_title, level = 1) short_title: 传一个channel实例的short_title值，如get_menu('product')
level: 显示的层级深度，默认为2级；如果要显示3级，则调用：get_menu('product', 3)
通过传入的父级channel，获取下拉菜单
get_slice_pages(pages, count) pages：一个Admin::Page数组，即页面数组
count: 分组数量
将pages数组按照每count一组分组，用于一些特殊展示， 例：[1,2,3,'a', 'b', 'c','d'] => [[1, 2, 3], ["a", "b", "c"], ["d"]]
get_images_from_content(content) content: 代表channel.content或page.content
把content内容里面的图片全部查询出来，包括宽高，最好返回一个图片数组； 例： => {src: "/ckeditor/pictures/148/original.jpg", alt: 'hello', width: "640", height: "427"}
