如何添加栏目

栏目（导航）是一个网站的核心框架，每个栏目的模板可以独立设计，以实现网站展示效果的多样性。

初始化栏目可以在db/seeds.rb文件中完成，如下所示：

    Admin::Channel.create!(
      :parent_id    => nil,
      :typo         => 'article',
      :title        => '首页',
      :short_title  => 'index',
      :tmp_index    => 'temp_index.html',
      :tmp_detail   => 'temp_detail.html',
      :keywords     => '首页',
      :description  => '欢迎使用RainCMS, 成都探码科技有限公司出品！'
    )

    Admin::Channel.create!(
      :parent_id    => nil,
      :typo         => 'product',
      :title        => '博客',
      :short_title  => 'blog',
      :tmp_index    => 'temp_blog.html',
      :tmp_detail   => 'temp_detail.html',
      :keywords     => '博客列表',
      :description  => ''
    )

也可以直接登录到后台‘栏目管理’中创建栏目。

属性说明：

属性说明
parent_id 父级栏目ID，顶级栏目则不填
typo 有三个值：article image product，分别表示这个栏目是用于文本链接、图片展示、还是商品
title 栏目名称
short_title 栏目URL缩写，如http://localhost:3000/blog/23
properties (废弃)
default_url (废弃)
tmp_index 栏目模板
tmp_detail 栏目下的子页面模板
keywords SEO关键词
description SEO描述信息
